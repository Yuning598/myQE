# Core and Walrasian equilibrium

导航：[Microeconomic index](../index.md)

Main note: [Microeconomic/03_General_Equilibrium_Production_and_Welfare](../03_General_Equilibrium_Production_and_Welfare.md)

A core allocation cannot be blocked by any coalition using only its own endowment.

Walrasian equilibrium allocations are in the core because any blocking coalition would need to buy bundles that are collectively more expensive than its endowment, contradicting feasibility.

Core convergence: in replica economies, core allocations converge to Walrasian allocations as market size grows.
