# FWT vs SWT assumptions

导航：[Microeconomic index](../index.md)

Main note: [Microeconomic/03_General_Equilibrium_Production_and_Welfare](../03_General_Equilibrium_Production_and_Welfare.md)

FWT: competitive equilibrium allocation is Pareto efficient.

Core assumptions:
- price-taking;
- complete markets for the commodities in the model;
- local nonsatiation/monotonicity;
- no externalities.

SWT: every Pareto efficient allocation can be decentralized after redistribution.

Additional assumptions:
- convex preferences;
- convex production sets;
- interiority/non-satiation;
- supporting hyperplane exists.

Trap: FWT is easier; SWT is where convexity is crucial.
