# 连续时间组合选择问题

:::{admonition} Definition (Portfolio Choice Problem)
连续时间组合选择问题是在给定 state variables $X_t$ 的条件下，同时选择 consumption policy 和 portfolio policy，使终身期望效用最大。

:::

资产价格过程通常写成

$$
\begin{aligned}
dB_t&=r(X_t,t)B_t\,dt,\\
\frac{dS_{n,t}}{S_{n,t}}
&=\mu_n(X_t,t)\,dt+\sum_{i=1}^d\sigma_{ni}(X_t,t)\,dZ_{i,t}.
\end{aligned}
$$

wealth dynamics 写成

$$
\begin{aligned}
dW_t
=\big(W_t r(X_t,t)+W_t\phi_t(\mu(X_t,t)-r(X_t,t)\mathbf 1)-c_t\big)\,dt
+W_t\phi_t\sigma(X_t,t)\,dZ_t.
\end{aligned}
$$

投资者问题是

$$
\begin{aligned}
P_t:\quad
\max_{c_s,\phi_s}\;
E_t\!\left[\int_t^T u_s(c_s)\,ds+U_T(C_T)\right].
\end{aligned}
$$

## 含义

它把连续时间最优决策拆成三部分：
- state variables $X_t$
- control variables $c_t,\phi_t$
- value function $V(t,W_t,X_t)$

这正是 HJB equation 的起点。
