# 价值函数 (Value Function)

导航：[Asset Pricing index](../../../index.md) · [Theoretical AP](../../../Theoretical_Asset_Pricing.md) · 来源：[02_Dynamic_Asset_Pricing](../../02_Dynamic_Asset_Pricing.md)


:::{admonition} Definition (Definition)
value function $V(t,x)$ 表示：在时点 $t$、给定当前状态 $x$ 时，从此刻起在所有可行策略中能达到的最大期望效用。

在当前讲义的消费—投资例子里，它就是

$$
\begin{aligned}
V(W_t)
:=\sup_{\{C_s,\phi_s\}_{s\ge t}}
E_t\!\left[\int_t^\infty e^{-\beta(s-t)}\log C_s\,ds\right].
\end{aligned}
$$
:::

(def-value-function)=

:::{admonition} Note
含义
它把“未来所有最优决策”压缩成“当前状态到一个标量数值”的映射；状态给定后，后续路径的最优值就由 $V$ 概括。
:::
