检验规则： Tn > c 时拒绝 H0。
Type I error 是 H0 为真却拒绝： α = P(Tn > c | H0)，即蓝色右尾面积。
Type II error 是 H1 为真却没有拒绝： β = P(Tn ≤ c | H1)，即绿色左侧面积。
Power 是正确拒绝假的 H0： 1−β = P(Tn > c | H1)。
