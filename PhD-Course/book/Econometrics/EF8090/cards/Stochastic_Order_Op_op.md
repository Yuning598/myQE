# Stochastic Order: Op and op

Parent: [04_Asymptotic_Tools](04_Asymptotic_Tools)

:::{admonition} Definition (Stochastic order)
\[
X_n=o_p(1)\iff X_n\to_p0,
\qquad
X_n=O_p(1)\iff X_n\text{ is bounded in probability}.
\]

**Lemma:** Useful algebra
\[
o_p(1)+o_p(1)=o_p(1),
\quad
O_p(1)+o_p(1)=O_p(1),
\quad
O_p(1)o_p(1)=o_p(1).
\]

:::

This notation keeps asymptotic proofs compact.
