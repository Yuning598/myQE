# DiD Common Trends

Parent: [12_DiD_Fixed_Effects_Event_Study](12_DiD_Fixed_Effects_Event_Study)

:::{admonition} Definition (Common trends)
\[
E[Y_0(post)-Y_0(pre)\mid D=1]
=E[Y_0(post)-Y_0(pre)\mid D=0].
\]

**Lemma:** 2x2 DiD
\[
\tau=(\bar Y_{1,post}-\bar Y_{1,pre})-(\bar Y_{0,post}-\bar Y_{0,pre}).
\]

:::

Common trends is about untreated potential outcomes, not observed outcomes after treatment.
