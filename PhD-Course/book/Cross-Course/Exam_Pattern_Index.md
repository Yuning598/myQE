# Exam Pattern Index

## 1. Proof-based

常见任务：

- 证明无套利等价于存在正状态价格。
- 证明 SDF 线性推出 CAPM。
- 证明 Jensen、风险厌恶、凹性的等价关系。
- 证明 moment condition 识别参数或 overidentifying restrictions。

当前 bridge：

- [State prices, SDF, and complete markets](bridges/State_Prices_SDF_Complete_Markets.md)

## 2. Computational

常见任务：

- 给 payoff matrix，算 rank 并判断市场是否完备。
- 给资产价格，求 state price set。
- 给消费增长和效用参数，算 Arrow price、bond price、risk-free rate。
- 给矩条件和权重矩阵，算 GMM estimator 或 asymptotic variance。

当前 bridge：

- [State prices, SDF, and complete markets](bridges/State_Prices_SDF_Complete_Markets.md)

## 3. Construction

常见任务：

- 构造套利组合。
- 构造复制组合。
- 构造 counterexample 说明命题的边界条件。

当前 bridge：

- [State prices, SDF, and complete markets](bridges/State_Prices_SDF_Complete_Markets.md)

## 4. Interpretation

常见任务：

- 解释 incomplete markets 为什么导致 state price 不唯一。
- 解释 SDF、state price、risk-neutral probability 的区别。
- 解释 empirical corporate finance 里的 identification threat。

当前 bridge：

- [State prices, SDF, and complete markets](bridges/State_Prices_SDF_Complete_Markets.md)
