# Reading Note: Vermaelen (1984)

## 1. Core Question
Vermaelen asks how repurchase tender offers can credibly signal undervaluation and how managerial incentives affect the credibility of that signal.

## 2. Core Logic
- repurchase is a **non-dissipative signal**: the main cost often shows up as wealth transfer through a premium rather than pure deadweight loss
- the signal is more credible when managers have more skin in the game or when the premium / repurchase commitment is costly to mimic
