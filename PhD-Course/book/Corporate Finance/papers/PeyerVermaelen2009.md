# Reading Note: Peyer and Vermaelen (2009)

[Back to the buyback framework](../01_Empirical_Corporate_Finance.md#sec-buyback-theories-framework)

## 1. Core Question
Which buyback characteristics predict stronger and more persistent post-buyback abnormal returns?

## 2. Main Point
- Buyback signal credibility is not binary.
- Cheap-stock features such as low market-to-book, small size, prior price decline, and explicit undervaluation statements are associated with stronger buyback anomalies.
- An undervaluation index can summarize these features and predict post-buyback performance.
