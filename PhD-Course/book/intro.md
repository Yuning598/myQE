# PhD Course Review Notes

This book is a PhD course review system for asset pricing, corporate finance, econometrics, and microeconomics. It is organized for three uses: reading course notes in order, drilling problem-set solutions, and translating the same mathematical restriction across courses.

## Course Map

- **Asset Pricing** connects SDF, beta pricing, dynamic pricing, options, beliefs, and empirical return tests.
- **Corporate Finance** connects capital structure, payout policy, corporate events, empirical design, and paper-based mechanisms.
- **Econometrics** supplies projection, IV, GMM, DiD, RD, panel methods, likelihood, and inference tools.
- **Microeconomics** supplies preferences, uncertainty, equilibrium, welfare, state prices, and information constraints.

## Recommended Flow

1. Start from the relevant course index and follow its chapter sequence.
2. Use cards when a chapter links to a concept that needs a precise definition or proof.
3. Switch to Problem Sets when the target is exam writing, calculation, or proof recall.
4. Use Cross-Course Links when one object appears under different names, such as SDF, projection, moment restriction, identification, or HJB.

## Reading Rule

The shortest path is:

$$
\begin{aligned}
\text{course index}
&\Longrightarrow \text{chapter note}
\Longrightarrow \text{card / bridge note}
\Longrightarrow \text{problem-set proof template}.
\end{aligned}
$$
